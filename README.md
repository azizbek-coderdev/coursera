 # Simple Interest Calculator

This project is a simple Bash script that calculates the **Simple Interest** based on the formula:

\[
Simple\ Interest = \frac{Principal \times Rate \times Time}{100}
\]

The project is part of the **Coursera Git and GitHub hands-on assignment** and follows the required repository structure and rules.
